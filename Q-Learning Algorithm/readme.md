# ***Q-Learning Algorithm***   

- ***Model-Free***  
  কিছুই জানে না—শুধু experience থেকে শেখে
- ***Value-Based***  
  এরা শিখে, তারপর policy বানায় মানে, “এই state কতটা ভালো—তা শিখে action নেবো”
- ***Off-Policy***   
  যে policy দিয়ে action নেওয়া হয়, সেই policy থেকে আলাদা policy-কে evaluate ও learn করা হয়।
